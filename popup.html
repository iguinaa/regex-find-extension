Regex Find

Press Ctrl+Shift+F to open regex search on any page.

Examples:

Simple text:
TODO → Find "TODO"
function → Find "function"

Wildcards:
test* → "test" + anything after
*ing → Anything + "ing"
f?x → "f" + any single char + "x"

Numbers:
\d+ → Any numbers
\$\d+ → Prices like $25

Ending matches:
Lorem\b → "Lorem" at word end
test[ .] → "test" + space or period
word$ → "word" at line end

Excluding chars:
cat [^s] → "cat" not followed by "s"
/ Imp [^VMB] → "/ Imp " + not V/M/B

Tip: Matches automatically stop at line breaks for easier reading.
